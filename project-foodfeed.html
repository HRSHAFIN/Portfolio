FoodFeed

2024 Mobile App (Android) Flutter Developer

Project Overview

FoodFeed is a hybrid social-commerce application built with Flutter, designed for the modern foodie community. It bridges the gap between food discovery and social interaction, allowing users to browse marketplace items, share personal reviews, and engage with others in real-time. The application focuses on a high-fidelity user experience with custom brand aesthetics and fluid multi-page navigation.

The Challenge

Building a media-heavy social application required efficient handling of asynchronous operations. The primary technical hurdle was implementing a smooth image picking and uploading workflow while maintaining app performance. Additionally, I had to architect a real-time-style Chat UI that felt responsive, managing state across multiple nested widgets without compromising the frame rate.

Key Features

  • Social Ecosystem: Dynamic feed area where users post food reviews with high-quality images and descriptions.
  • Real-time Messaging: Interactive chat interface with active status indicators and modern message bubble logic.
  • Media Integration: Seamless hardware access using image_picker for gallery selection and camera capture.
  • Interactive Detail Pages: Dedicated post views with functional 'Like' and 'Comment' systems to drive user engagement.
  • Advanced Navigation: Multi-layered UX utilizing custom Side Drawers and Bottom Navigation Bars.
