Project Overview

This comprehensive Machine Learning Pipeline was developed to analyze and predict patterns in motor vehicle crashes. By integrating classification, regression, and unsupervised learning techniques, the system identifies high-risk boroughs, predicts the severity of injuries, and clusters accidents based on contributing factors. The project demonstrates an end-to-end workflow from advanced data preprocessing to hyperparameter optimization.

The Challenge

Handling large-scale, high-dimensional datasets required rigorous feature engineering and selection. Key challenges included managing categorical data from different boroughs (Bronx, Brooklyn, Manhattan, etc.), optimizing models for high-variance targets like injury counts, and ensuring cluster stability across unsupervised models. Balancing model precision with computational efficiency was critical for processing thousands of crash records.

Pipeline Architecture & Features

  • Classification Module: Implemented XGBoost and Random Forest to predict injury flags with automated hyperparameter tuning using GridSearchCV.
  • Regression Module: Developed models to estimate the number of persons injured, utilizing StandardScaler pipelines for optimal feature convergence.
  • Unsupervised Clustering: Applied K-Means Clustering to identify distinct accident profiles, optimized via the Elbow Method and Inertia analysis.
  • Automated Preprocessing: Integrated robust pipelines for feature scaling, outlier handling, and categorical encoding (One-Hot Encoding).
  • Optimization Strategy: Leveraged Optuna and Randomized Search for advanced model selection and validation.
