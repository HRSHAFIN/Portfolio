Hishab Nikash

2025 FinTech / Web App Frontend Architect

Project Overview

Hishab Nikash is a lightweight, intuitive web application architected for personal and small business financial tracking. It provides a seamless interface to manage client accounts, log intricate daily transactions, and monitor multiple money streams like Bank vs. Cash balances. Designed as a complete frontend prototype, the system leverages local storage for instant performance and zero-latency data handling.

The Challenge

The primary challenge was creating a complex accounting system that felt fast and responsive without a heavy backend database. I had to design a robust data structure using localStorage that could handle relational-style data (connecting transactions to specific accounts) while ensuring the UI remained synchronized during bulk CSV imports and real-time filtering.

Key Features

  • Advanced Account Management: Full CRUD functionality for managing user and company financial profiles.
  • Dual-Stream Tracking: Separate ledgers for Bank and Cash transactions with automated net balance calculations.
  • Data Visualization: Integrated Chart.js dashboard providing bar and line graph analysis of financial health.
  • CSV Portability: Native JS implementation for bulk importing and exporting data as .csv files for external reporting.
  • Premium UX: GSAP and Anime.js powered pre-loaders and glass-morphism login portals for a modern aesthetic.
